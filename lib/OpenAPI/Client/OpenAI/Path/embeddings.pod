=head1 NAME

OpenAPI::Client::OpenAI::Path::embeddings - Documentation for the /embeddings path.

=head1 DESCRIPTION

This document describes the API endpoint at C</embeddings>.

See the C<examples/> directory in the distribution for examples of how to use this.

=head2 C<POST /embeddings>

Creates an embedding vector representing the input text.

=head3 Operation ID

C<createEmbedding>

    $client->createEmbedding( ... );


=head3 Request Body

=head3 Content Type: C<application/json>


=head3 Example:

    {
       "user" : "user-1234",
       "model" : "text-embedding-3-small",
       "encoding_format" : "float",
       "input" : "The quick brown fox jumped over the lazy dog"
    }



=head3 Responses

=head3 Status Code: C<200>

OK

Content Types:

=over 4

=item * C<application/json>

Example:

    {
       "usage" : null,
       "data" : [
          {
             "embedding" : [
                null
             ]
          }
       ]
    }



=back
